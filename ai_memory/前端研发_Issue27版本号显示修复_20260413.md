# 前端研发 - Issue#27 版本号显示修复

## 标签

前端, Mia, Issue#27, App.vue, 版本号, getVersion, AppVersion, v1.2.0

## 任务概述

修复 Issue#27: UI 未显示系统版本号。在 Header 中动态展示从后端 `/api/v1/version` 获取的版本号。

## 变更文件清单

| 文件 | 变更内容 |
| --- | --- |
| `frontend/src/api/types.ts` | 新增 `AppVersion` 接口 `{ version: string }` |
| `frontend/src/api/index.ts` | import 追加 `AppVersion`; 新增 `getVersion()` 函数 (GET /version) |
| `frontend/src/App.vue` | `onMounted` 调用 `getVersion()`; Header 显示 `v{{ appVersion }}`; 失败静默处理; 新增 `.app-version` 样式 |
| `frontend/package.json` | version: "1.0.0" -> "1.2.0" |
| `frontend/dist/` | 重新构建产物 |

## 关键实现细节

- `getVersion()` 请求路径为 `/version`，baseURL 已配置为 `/api/v1`，实际请求 `/api/v1/version`
- 版本号获取失败时 `catch {}` 静默处理，不影响主功能
- `v-if="appVersion"` 控制渲染，未获取到时不显示版本号元素
- `.app-version` 样式: `color: rgba(255,255,255,0.75)`, `font-size: 0.8rem`, `margin-left: 10px`
