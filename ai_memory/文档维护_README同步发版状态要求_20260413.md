# 文档维护 - README 同步发版状态要求

**标签**: README, 发版管理, 文档维护

## 内容

项目经理明确要求：每次做发版准备时必须同步维护 /README.md，确保其内容与当前发版状态保持一致。

## README.md 维护要点

每次发版时 README.md 应同步更新以下内容：

1. **当前版本号**：更新 "当前版本" 字段为本次发布版本；
2. **文档版本引用**：更新目录结构图中的文档版本号（pdd/sad/tsr/tbp 等）；
3. **部署方式说明**：如部署方式有重大变更（如 v1.1.0 放弃 Docker），同步更新快速开始章节；
4. **目录结构**：如有新增/删除重要目录或文件，同步更新目录结构图；
5. **文档引用**：发布说明文件路径（release_notes_vX.X.X.md）更新为最新版本。
