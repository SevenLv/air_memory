## 关键实现

### formatLocalTime 逻辑

```typescript
export function formatLocalTime(dateStr: string): string {
  if (!dateStr) return ''
  const date = new Date(dateStr)
  if (isNaN(date.getTime())) return dateStr
  const pad = (n: number): string => String(n).padStart(2, '0')
  // 使用 getFullYear/getMonth 等本地时间方法（非 UTC 方法）
  return `${date.getFullYear()}-${pad(date.getMonth() + 1)}-${pad(date.getDate())} ${pad(date.getHours())}:${pad(date.getMinutes())}:${pad(date.getSeconds())}`
}
```

### 时间列渲染模式（prop 改为 slot）

由于使用 `formatLocalTime()` 函数，`el-table-column` 不能使用 `prop="created_at"` 直接渲染，
需改为自定义 slot：

```html
<el-table-column label="时间" width="200">
  <template #default="{ row }">
    {{ formatLocalTime(row.created_at) }}
  </template>
</el-table-column>
```

### 测试更新要点

`MemoryCard.spec.ts` 中的创建时间测试由：
```typescript
expect(wrapper.text()).toContain(coldMemory.created_at)
```
改为：
```typescript
expect(wrapper.text()).toMatch(/\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}/)
```
