## 关键信息

### 部署手册核心内容

- 环境要求：Docker Engine 27+ 或 Docker Desktop，8GB 内存，50GB 磁盘
- 首次构建需访问 HuggingFace Hub 下载 all-MiniLM-L6-v2 模型（约 90MB）
- macOS 一键启动：`bash start.sh`
- Windows 一键启动：双击 `start.bat` 或命令行执行
- 部署验证：访问 `http://localhost:8080`，`curl http://localhost:8080/health` 返回 `{"status": "ok"}`

### 用户手册核心内容

- Web 界面：4 个页面（`/` 查询、`/memories` 管理、`/logs` 日志、`/feedback` 价值评分）
- MCP 端点：`http://localhost:8080/mcp`（3 个工具：save_memory, query_memory, feedback_memory）
- REST API 基础 URL：`http://localhost:8080/api/v1`
- API 文档：`http://localhost:8080/api/v1/docs`
