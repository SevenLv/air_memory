## Issue #46 根因与修复（Lydia 分析）

根因 1: start.bat/start.sh 中 uvicorn 命令缺少 --log-level warning，uvicorn.error 在
MCP SSE 连接断开重连时周期性输出 INFO 日志，触发 Windows FlashWindowEx。

根因 2: main.py 的日志抑制列表缺少 anyio 及其子模块，anyio 有独立 StreamHandler。

修复:
- start.bat 和 start.sh：uvicorn 命令添加 --log-level warning
- main.py：抑制列表补充 anyio、anyio._backends._asyncio、anyio._backends._trio

## Issue #47 根因与修复

根因: FastAPI 默认 ensure_ascii=True，中文变为 \uXXXX。

修复:
- 添加 orjson>=3.9 依赖
- main.py 设置 default_response_class=ORJSONResponse

## Issue #45 根因与修复

根因: 前端直接显示 UTC ISO 8601 字符串，未做时区转换。

修复:
- 新建 frontend/src/utils/time.ts，提供 formatLocalTime() 函数
- 在 LogsView.vue、FeedbackView.vue、MemoryCard.vue 应用
