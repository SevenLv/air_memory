### v1.2.10 变更内容
- 记忆管理页面新增删除操作（确认弹窗 + 删除成功本地移除）
- 列表默认过滤已删除记忆（`memory_deleted=true`）
- 新增评价值列，通过 `save_logs LEFT JOIN memory_values` 返回 `value_score`
- 用户手册（doc/user_guide.md）v1.7 同步更新

### 发版操作最短路径
1. 更新版本号（6 个文件）：`backend/pyproject.toml`、`backend/src/air_memory/main.py`、`frontend/package.json`、`frontend/package-lock.json`、`start.sh`、`start.bat`
2. 更新测试版本断言：`backend/tests/test_main.py`（函数名 + 断言值）
3. 构建前端：`cd frontend && npm run build`（自动替换 dist/assets/ 下旧文件）
4. 新建发布说明：`doc/release_notes_v1.2.10.md`
5. 更新 README.md：当前版本号 + Mermaid 图索引
6. 更新工作流：`.github/workflows/release.yml` 中 `--notes-file` 指向新版本
7. 提交推送，在 GitHub Actions 手动触发 release.yml，输入 `v1.2.10`
