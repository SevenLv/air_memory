### 任务概要

执行 M1 阶段（后端服务就绪）全部 9 项工作条目，并输出 M1 阶段研发执行报告。

### 关键结论

- M1-01 至 M1-09 全部完成，验收标准 M1-AC-01 至 M1-AC-13 全部达成；
- 后端代码位于 `backend/src/air_memory/`，目录结构：config.py, db.py, main.py, api/, memory/, feedback/, disk/, log/, mcp/, models/；
- 所有性能阈值通过 `config.py` 的 Settings 类集中管理，支持环境变量覆盖；
- 热层=EphemeralClient（纯内存），冷层=PersistentClient（磁盘，路径由 CHROMA_COLD_PATH 配置）；
- SQLite 路径由 DB_PATH 配置，默认 `./data/logs.db`；
- MCP Server 挂载于 `/mcp`，使用 FastMCP.streamable_http_app()；
- 日志写入和层间迁移均使用 asyncio.create_task() 异步执行，不阻塞主业务；
- M1 研发执行报告：`/doc/m1_report_v1.0.md`。
