# 记忆管理 UI - 删除过滤与评价值展示最短路径

## 正文
1. 先定位 `frontend/src/views/MemoriesView.vue`，该页数据源为 `GET /api/v1/logs/save`。
2. 后端最小改动：`backend/src/air_memory/log/service.py` 对 `save_logs` 与 `memory_values` 做 LEFT JOIN，返回 `value_score`；`backend/src/air_memory/models/log.py` 的 `SaveLog` 增加 `value_score`。
3. 前端最小改动：
   - 列表新增“评价值”列，格式化两位小数；
   - `filteredLogs` 中过滤 `memory_deleted=true`；
   - “操作”列新增“删除”按钮，调用 `DELETE /api/v1/memories/{id}`，成功后本地移除；
   - 增加删除确认弹窗，避免误删。
4. 测试最小补充：
   - `frontend/tests/MemoriesView.spec.ts` 增加删除行为与删除过滤、评价值展示断言；
   - `backend/tests/test_log_service.py` 增加 value_score 回归测试（固定时间戳）。
5. 文档同步：`doc/user_guide.md` 的“2.3 记忆管理”补充删除操作、删除过滤、评价值列说明。
6. 验证顺序：先定向测试，再前端全量测试+构建，再后端全量 pytest。
