## 本地复现打包结果

复现命令（基于 workflow 逻辑）：

```bash
TAG="v1.2.6"
PKG_NAME="air_memory-${TAG}"
PKG_DIR="${PKG_NAME}"

mkdir -p "${PKG_DIR}/backend"
mkdir -p "${PKG_DIR}/frontend"
cp README.md start.sh start.bat "${PKG_DIR}/"
cp -r doc "${PKG_DIR}/"
cp backend/pyproject.toml backend/requirements.txt "${PKG_DIR}/backend/"
cp -r backend/src "${PKG_DIR}/backend/"
cp -r frontend/dist "${PKG_DIR}/frontend/"
zip -r "${PKG_NAME}.zip" "${PKG_DIR}/"
```

生成的 zip 包验证：
- zip 文件名：`air_memory-v1.2.6.zip` ✅
- 内部顶级目录：`air_memory-v1.2.6/` ✅
- backend 内容：src/ + pyproject.toml + requirements.txt（无 tests/ 和 Dockerfile）✅
- frontend 内容：dist/assets/ + dist/index.html（无 src/ 和 tests/）✅
- doc 内容：全部 doc/ 文档（含 deploy_guide.md、user_guide.md 等）✅
