# 发布管理_v1.0.4发布准备_20260410

## 标签

发布, v1.0.4, Hotfix, release_notes, GitHub Release, start.bat, Windows 10 1809

## 发布背景

v1.0.4 为 v1.0.3 的 Hotfix 版本. 修复内容: Windows 10 1809 (Build 17763) 上执行 `start.bat` 时, 脚本打印标题横幅后立即崩溃并报错 `需安装兼容版本 was unexpected at this time.`. 根因为 `start.bat` 第 28 行位于嵌套 `if ( ... )` 复合块内, Windows 10 1809 的 CMD 解析器在处理包含多字节中文字符的 `echo` 行时对括号深度计数出现偏差, 导致 `)` 过早关闭内层 `if` 块, 触发解析错误. 修复方案为将第 28 行括号改为 `^(` / `^)` 转义.
