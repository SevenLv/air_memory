3. **deploy_guide.md 3.3 节**：v1.2.0 在 `start.bat` 启动成功 Banner 中新增"后端 API 文档"行，但文档未同步更新。

## 关键知识点

- MCP `query_memory` 实际返回：`json.dumps([m.model_dump() for m in memories], ensure_ascii=False)`，即 JSON 字符串格式的平铺列表，与 REST API 响应结构不同。
- REST API `GET /api/v1/memories` 返回：`{"memories": [...], "count": N, "query_mode": "fast"/"deep"}`。
- 当前文档最新版本：user_guide.md=v1.1, deploy_guide.md=v1.2。
