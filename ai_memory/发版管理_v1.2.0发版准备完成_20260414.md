# AIR_Memory v1.2.0 发版管理记忆

## 发版背景

Issues #28 (start.bat UI 地址 http://0.0.0.0:8080 浏览器无法访问) 和 #27 (UI 和启动脚本未显示版本号) 修复后发布 v1.2.0.

## 修复内容

- Issue #28: uvicorn --host 改为 127.0.0.1
- Issue #27: 后端新增 APP_VERSION 常量和 /api/v1/version 端点, 前端 Header 动态显示版本, 启动脚本 Banner 显示版本

## 发版流程

1. 创建 doc/release_notes_v1.2.0.md
2. 更新 README.md 版本号
3. 更新 .github/workflows/release.yml 指向新 release notes
4. 合并 PR 后通过 workflow_dispatch 触发 release.yml, 输入 tag=v1.2.0

## 注意

- release.yml 使用 workflow_dispatch 手动触发
- 触发时在 GitHub Actions 界面输入版本号 v1.2.0
