## 关键注意事项

### start.bat 编码问题

start.bat 文件存在编码问题，必须使用 byte-safe 方式读写:

```python
with open('start.bat', 'rb') as f:
    content = f.read()
content = content.replace(b'--host 0.0.0.0', b'--host 127.0.0.1')
old_banner = 'echo  AIR_Memory 一键启动\r\n'.encode('utf-8')
new_banner = 'echo  AIR_Memory 一键启动 v1.2.0\r\n'.encode('utf-8')
content = content.replace(old_banner, new_banner)
with open('start.bat', 'wb') as f:
    f.write(content)
```

- 必须保持 CRLF 行尾 (文件内有 \r\n)
- 禁止使用全角中文符号
- 禁止使用文本编辑器直接修改 (会破坏编码)

### /api/v1/version 端点位置

- 路由前缀 `/api/v1` 已在 `router = APIRouter(prefix="/api/v1")` 中定义
- 若通过 router 添加，路径会变为 `/api/v1/api/v1/version` (重复前缀)
- 正确做法：直接在 app 上定义，指定完整路径 `/api/v1/version`
- 与 `/health` 端点位置相同 (在 StaticFiles mount 之前)

### APP_VERSION 常量位置

- 在文件顶部 import 区块之后、lifespan 函数定义之前定义
- FastAPI 初始化使用 `version=APP_VERSION` 引用
