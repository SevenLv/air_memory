## 修改文件清单

### Task 1 — Issue #42：版本号升级（1.2.2 -> 1.2.4）

| 文件 | 修改内容 |
|------|----------|
| `backend/src/air_memory/main.py` | `APP_VERSION = "1.2.2"` -> `"1.2.4"` |
| `backend/pyproject.toml` | `version = "1.2.2"` -> `"1.2.4"` |
| `start.sh` | 第 79 行 banner 版本号 v1.2.2 -> v1.2.4 |
| `start.bat` | **必须使用 Python bytes 方式**替换，禁止文本编辑器（UTF-8 BOM + CRLF） |

start.bat 修改方式：
```python
with open('start.bat', 'rb') as f:
    content = f.read()
content = content.replace(b'v1.2.2', b'v1.2.4')
with open('start.bat', 'wb') as f:
    f.write(content)
```

## 注意事项

- start.bat 为 UTF-8 BOM + CRLF 编码，必须用 Python bytes 方式修改，禁止 text editor 直接写

- `get_all_feedback_logs` 使用 f-string 构建动态 SQL，params 为列表（支持追加 page_size 和 offset）
