# 后端研发 - 静态文件服务与 CORS 更新

**标签**：#后端研发 #FastAPI #静态文件 #CORS #SPA #main.py #env_config

## 任务概述

架构从 Docker 部署调整为 Python 本机直接运行。FastAPI 统一在端口 8080 提供 REST API、MCP 服务和前端静态文件服务。

## 变更文件

### 1. `backend/src/air_memory/main.py`

**关键变更**：
- 添加 `from fastapi.staticfiles import StaticFiles` 导入
- CORS 配置改为从 `CORS_ORIGINS` 环境变量读取（逗号分隔，默认 `http://localhost:8080,http://127.0.0.1:8080`）
- 在所有 API 路由注册之后，添加静态文件挂载（仅当目录存在时生效）
- 使用 `StaticFiles(html=True)` 实现 SPA 路由回退（内置，无需额外 catch-all 路由）
- 更新 docstring 说明默认端口为 8080

**关键注意事项**：
- `StaticFiles` 挂载必须在所有 API 路由（include_router、mount("/mcp")、/health）注册之后
- **不要**同时添加 `/{full_path:path}` catch-all 路由，会导致静态文件挂载不可达
- `html=True` 已内置 SPA 回退，找不到文件时自动返回 index.html
- 使用条件判断 `if os.path.isdir(STATIC_DIR)` 确保测试环境不受影响

### 2. `doc/env_config.md`

新增配置项：

| 变量名 | 默认值 | 说明 |
| --- | --- | --- |
| `PORT` | `8080` | 服务监听端口（由启动脚本通过 uvicorn 命令行参数设置） |
| `STATIC_DIR` | `./frontend/dist` | 前端静态文件目录（相对于工作目录） |
| `CORS_ORIGINS` | `http://localhost:8080,http://127.0.0.1:8080` | 允许的 CORS 来源（逗号分隔） |

去除 Docker 容器路径相关说明，更新默认存储路径为本机相对路径。
