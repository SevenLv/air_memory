# 前端研发 - Issue UI 时间本地化显示

## 标签

前端, 时间, 本地化, formatLocalTime, Vue, TypeScript

## 内容

### 问题

UI 上的时间类信息显示的是后端返回的原始时间字符串（服务器时间），需改为显示用户本地时间。

### 技术方案

- 新建 `frontend/src/utils/time.ts`，提供 `formatLocalTime()` 工具函数
- 使用 JS 原生 `Date` 对象，无需第三方库
- 将 ISO 8601 时间字符串转换为本地时区的 `YYYY-MM-DD HH:mm:ss` 格式

### 修改文件
