### 部署架构

- backend 服务：端口 8000（内部），使用命名 Volume `/app/data`
- frontend 服务：端口 8080（对外），Nginx 反向代理 `/api/` 至 backend
- 内部网络：`air_memory_net`（bridge 模式）

### 最短路径（快速复现 M4）

1. 构建 frontend 镜像：`docker compose build frontend`（无网络限制，直接成功）
2. 构建 backend 镜像（有外网）：`docker compose build backend`
3. 构建 backend 镜像（无外网/CI）：`docker compose build --build-arg PREDOWNLOAD_MODEL=false backend`
4. 启动服务：`docker compose up -d`
5. 验证：`curl http://localhost:8080/`（返回 200）
