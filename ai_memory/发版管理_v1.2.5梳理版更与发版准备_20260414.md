## 任务背景

## v1.2.4 之后变更梳理

v1.2.4 发布后，合并了以下 3 个修复：

### Issue #45 - UI 时间字段显示 UTC 而非本地时间

- 新建 `frontend/src/utils/time.ts`，提供 `formatLocalTime()` 工具函数
- 在 LogsView.vue、FeedbackView.vue、MemoryCard.vue 四处时间列应用本地时间转换

### Issue #46 - Windows 浏览器始终激活自身（彻底修复）

- start.bat / start.sh：uvicorn 命令增加 `--log-level warning`
- main.py：日志抑制列表补充 anyio 及其子模块

### Issue #47 - 存储操作日志乱码徽章检测失效 + F12 Unicode 转义

- 后端 `log/service.py`：提取 `_is_garbled()`，修复纯 ASCII 问号检测漏洞
- 后端 `models/log.py`：`SaveLog` 新增 `is_garbled: bool` 动态字段
- 前端 `LogsView.vue`：优先信任服务端 `is_garbled`，修复兜底检测
- 添加 `orjson>=3.9` 依赖，FastAPI 设置 `ORJSONResponse` 解决 Unicode 转义问题
